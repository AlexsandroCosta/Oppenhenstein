Teorias

O Princípio fundamental da contagem (P.F.C), é a ideia de que o número de possibilidades de fazer X ações distintas e independentes, é a multiplicação da quantidade de modos possíveis que cada uma pode ser feita. Ou seja, se uma decisão pode ser tomada de N maneiras e outra decisão pode ser tomada de M maneiras, o número de maneiras que essas decisões podem ser tomadas simultaneamente é calculado pelo produto de N · M.

Exemplo 1: Com 5 homens e 5 mulheres, de quantos modos se pode formar um casal?

Solução: Seguindo a lógica do P.F.C existem 5 maneiras de escolher um homem e 5 maneiras de escolher uma mulher.

Então há 55 = 25 modos de se formarem um casal. Pois existem 5 opções para escolher um par para cada pessoa, como nesse caso a escolha é não ordenada então a ordem de como os elementos são combinados não importa (Explicado no módulo 4.2), por exemplo, Maria e João é igual a João e Maria.

Não há uma solução única para todos os problemas, o princípio fundamental da contagem vai depender das situações únicas geradas por um problema, entre ordenado e não ordenado. Para facilitar a resolução de problemas pode-se seguir os seguintes passos como guia.

  1. Postura: Devemos sempre nos colocar no papel da pessoa que deve fazer a ação solicitada pelo problema e ver que decisões devemos tomar.
  2. Divisão: Devemos, sempre que possível, dividir as decisões a serem tomadas em decisões mais simples.
  3. Não adiar dificuldades: Pequenas dificuldades adiadas costumam se transformar em imensas dificuldades.

Exemplo 2: Quantos divisores inteiros e positivos possui o número 360? Quantos desses divisores são pares? Quantos são ímpares? Quantos são quadrados perfeitos?

Solução: Primeiramente iremos decompor o 360 utilizando M.M.C (Mínimo Múltiplo Comum) 360 = 23 · 32 · 5. Os divisores inteiros e positivos de 360 são os números da forma 2α · 3β · 5γ, com α ∈ = {0, 1, 2, 3}, β ∈ = {0, 1, 2} e γ ∈ = {0, 1}.

Depois vamos pegar quantidade de elementos de cada conjunto e multiplicarmos 4 · 3 · 2 = 24. 24 maneiras de escolher os expoentes α, β, γ. Assim temos 24 divisores inteiros e positivos de 360.

Agora vamos descobrir quantos desses divisores são pares, para isso o α não pode ser 0, então existem 3 · 3 · 2 = 18 divisores pares. E para a quantidade de ímpares α tem que ser 0, dessa forma há 1 · 3 · 2 = 6 divisores ímpares.

E por último vamos descobrir quantos são os quadrados perfeitos, para ser um quadrado perfeito os expoentes α, β e γ devem ser par. Há 2 · 2 · 1 = 4 divisores que são quadrados.

Há alguns problemas de Combinatória que, embora sejam aplicações do princípio básico, aparecem com muita frequência. Para esses problemas, vale a pena saber de cor as suas respostas.

O primeiro desses problemas é o:

Problema das permutações simples

De quantos modos podemos ordenar em fila N objetos distintos?

A escolha do objeto que ocupará o primeiro lugar pode ser feita de N modos; a escolha do objeto que ocupará o segundo lugar pode ser feita de N - 1 modos, e o terceiro N-2, e etc.

Portanto, o número de permutações simples de N objetos distintos, ou seja, o número de ordens em que podemos colocar N objetos distintos é P = n!.

Exemplo 1: De quantos modos podemos arrumar em fila 5 livros diferentes de Matemática, 3 livros diferentes de Estatística e 2 livros diferentes de Física, de modo que livros de uma mesma matéria permaneçam juntos?

Solução: Podemos escolher a ordem das matérias de 3! modos. Feito isso, há 5! modos de colocar os livros de Matemática nos lugares que lhe foram destinados, 3! modos para os de Estatística e 2! modos para os de Física.

A resposta é 3! · 5! · 3! · 2! = 6 · 120 · 6 · 2 = 8640.

O segundo problema importante é o:

Problema das combinações simples

De quantos modos podemos selecionar p objetos distintos entre n objetos distintos dados?

Cada seleção de P objetos é chamada de uma combinação simples de classe P dos N objetos, assim, por exemplo, as combinações simples de classe 3 dos objetos a, b, c, d, e são {a, b, c}, {a, b, d}, etc…

Representando assim por C n p ou ( p n ) , deixando assim C 5 3 = ( 3 5 ) . Agora basta apenas aplicar a equação para combinações desse tipo:

C n p = n! p!(n-p)!

Exemplo 2: Com 5 homens e 4 mulheres, quantas comissões de 5 pessoas, com exatamente 3 homens, podem ser formadas?

Solução: Para formar a comissão devemos escolher 3 dos homens e 2 das mulheres. Há C 5 3 · C 4 2 = 10 · 6 = 60 comissões.

Respectivamente o primeiro problema funcionará melhor em problemas que envolvem uma ordem exata, ou problemas ordenados. Para problemas que não exigem ordenação, problemas não ordenados, o conceito do segundo problema exibido exerce melhor utilidade.

Em problemas mais complexos vale levar em conta que se tivermos duas ou mais decisões (e) considera-se multiplicação, já se tivermos que escolher uma ou outra (ou) considera-se a soma.

A fórmula do binômio de Newton é a fórmula que dá o desenvolvimento de (a + b)n, conhecido como desenvolvimento binomial, servindo para determinar a solução de um binômio elevado a enésima potência.

O termo genérico do produto é obtido tomando em p dos fatores, p = O, 1,2, . . . , n, na segunda parcela e tomando nos restantes n — p fatores a primeira parcela. Como isso pode ser feito de C n p , modos, o termo genérico do produto é C n p · an - p · bp assim:

(a + b)n: ∑ p = 0 n C n p · a n - p · b p ou ∑ p = 0 n ( p n ) · a n - p · b p

Exemplo 1: Determine o coeficiente de x3 no desenvolvimento de ( x 4 - 1 x ) 7 .

Solução: O termo genérico do desenvolvimento é:

C 7 p · ( x 4 ) 7 - p ( 1 x ) p = C 7 p · x 28 - 5p · (-1) p

O termo em x3 é obtido se 28 - 5p = 3, ou seja, se p =5.

O termo proucurado é C 7 5 · x 28 - 5 · 5 (-1) 5 = C 7 5 · x 3 · (-1) 5 = -21x 3 . Sendo assim o coeficiente é -21.
